# Ministry of Fun

A small collection of classic game clones created with [Godot 4.6](https://godotengine.org/).

This is inspired by the great [UFO 50](https://50games.fun/) but with a much smaller scope, and really just a learning project for myself to learn game development.

Right now this only contains a Snake clone, stay tuned for more!

## Controls

| Action | Keyboard | Controller | Mouse/Touch |
|:-------|:-------- |:---------- |:----------- |
| **Movement** | Arrow keys, WASD | D-Pad, left joystick | Swipe |
| **Pause** | Escape | Start | - |
| **Toggle fullscreen** | Alt-Enter | - | - |
| **Quit** | Ctrl-Q, Alt-F4 | - | - |
